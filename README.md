# FIFA-data-analysis
FIFA match data analysis using Python, Pandas, Matplotlib and Seaborn.

# FIFA Match Data Analysis ⚽📊

##  Project Overview
This project analyzes FIFA match data using Python to understand scoring patterns, match outcomes, and team performance.

The goal is to explore how home advantage affects match results and how teams perform in home vs away games.

##  Data Cleaning
- Handled missing values by filling or removing them
- Removed unnecessary columns (e.g. notes column)
- Checked and removed duplicates
- Converted data types where necessary

## Exploratory Data Analysis (EDA)

### Key calculations:
- Average goals per match: ~3.1
- Home team average goals: higher than away teams
- Away team average goals: lower overall

### Match outcome analysis:
- Home Win
- Away Win
- Draw

## Visualisations
- Histogram of total goals per match
- Count plot of match results
- Bar charts for team performance

##Key Insights
- Home teams score more goals on average
- Home teams win more matches (strong home advantage)
- Most matches have 0–3 total goals
- Only a few matches are high scoring (4+ goals)
- Certain teams consistently perform better at home or away

##  Skills Demonstrated
- Data cleaning (Pandas)
- Exploratory Data Analysis (EDA)
- Feature engineering (total goals, match results)
- Data visualization (Matplotlib, Seaborn)
- Grouping and aggregation

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

 Conclusion
This project demonstrates how football match data can be analyzed to identify patterns such as home advantage, scoring trends, and team performance differences.

It is a beginner-friendly end-to-end data analysis project.
